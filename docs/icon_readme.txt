# Icon Placeholder

Para generar el instalador con un icono personalizado:

1. Crear un archivo `icon.ico` en este directorio
2. Recomendaciones:
   - Tamaño: 256x256 píxeles
   - Formato: .ICO con múltiples resoluciones (16, 32, 48, 256)
   - Herramienta recomendada: https://www.icoconverter.com/

Si no existe `icon.ico`, el instalador usará el icono por defecto de Windows.
